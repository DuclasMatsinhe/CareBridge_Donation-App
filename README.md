# Care Bridge – Women Hygiene Donation App

## Project Overview
Care Bridge is a Human-Centered Design (HCD) based mobile application aimed at improving access to menstrual hygiene products for women in need. The platform connects donors with recipients through a simple, private, and efficient system.

The application allows users to request hygiene products and enables donors to contribute either products or funds, ensuring accessibility, dignity, and transparency.


## Problem Statement
Many women, especially in low-income communities, face difficulty accessing essential hygiene products due to financial constraints, lack of availability, and social stigma. Existing donation systems are not efficient or accessible enough to connect donors with those in need.

Care Bridge aims to bridge this gap through a digital solution.


## Features
- User Registration & Login  
- Role-based access (Donor / Recipient)  
- Request hygiene products  
- Donate products or funds  
- Anonymous request option  
- Donation tracking (planned)  
- User dashboard
